Kas ir HTML?

HTML (HyperText Markup Language) ir iezīmēšanas valoda, ko izmanto, lai izveidotu tīmekļa lapas. Tā nosaka lapas struktūru, izmantojot dažādus elementus un tagus. HTML tiek interpretēts pārlūkprogrammās, kas vizuāli attēlo lietotājiem tīmekļa saturu.

HTML pamatelementi

HTML dokumenti sastāv no dažādiem elementiem, piemēram:

  • <html> – apzīmē dokumenta sākumu un beigas.
  • <head> – satur meta informāciju, piemēram, lapas nosaukumu (<title>) un CSS stilus.
  • <body> – satur visu tīmekļa lapas galveno saturu, piemēram, virsrakstus (<h1>–<h6>), rindkopas (<p>), attēlus (<img>), saites (<a>) un sarakstus (<ul>, <ol>, <li>).

HTML atribūti

  • <a href="https://example.com">Apmeklē šo vietni</a> – href atribūts nosaka saites adresi.
  • <img src="attels.jpg" alt="Apraksts"> – src atribūts norāda attēla ceļu, bet alt nodrošina alternatīvo tekstu.
  • <input type="text" placeholder="Ievadi vārdu"> – placeholder atribūts rāda ievades lauka aprakstu.
